# ctags usage

* inside vim:

```
<Ctrl><]> 		jump to a symbol

<Ctrl><t> 		jump back

:ts <tag> <RET>		Search for a particular tag

:tn			Go to the next definition for the last tag

:tp			Go to the previous definition for the last tag

:ts			List all of the definitions of the last tag
```


* open vim to tag

```
vi -t <tagSTIRNG>
```

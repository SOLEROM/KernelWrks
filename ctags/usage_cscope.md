# cscope usage

```
search:
=======
 CTRL- \ + X		: search in the same window
 CTRL-spacebar +X 	: then a search type makes the vim window split horizontally
 CTRL-space *twice* +X 	: the search type does a vertical split instead of a horizontal one  

back:
=====
 CTRL-T                 : go back to where you were before the search.

```

## X key is: 

```
   's'   symbol: find all references to the token under cursor
   'g'   global: find global definition(s) of the token under cursor
   'c'   calls:  find all calls to the function name under cursor
   't'   text:   find all instances of the text under cursor
   'e'   egrep:  egrep search for the word under cursor
   'f'   file:   open the filename under cursor
   'i'   includes: find files that include the filename under cursor
   'd'   called: find functions that function under cursor calls

```
